# Design System Document

This document outlines the core design principles and token values for our digital products.

## Theme Overview
The current theme is set to `light` mode.

## Color Palette
- **Primary Color:** The brand's most distinctive chromatic color, suitable for buttons and CTAs. Currently set to `#006E2A`.
- **Secondary Color:** A supporting color for less prominent UI elements and secondary actions. Currently set to `#00C853`.
- **Tertiary Color:** An additional accent color for highlights or decorative elements. Currently set to `#C62828`.
- **Neutral Color:** A neutral base color for backgrounds and surfaces. Currently set to `#EEEEF0`.

## Typography
- **Headline Font:** `inter`
- **Body Font:** `inter`
- **Label Font:** `inter`

## Shape and Spacing
- **Roundedness:** UI elements feature moderate rounded corners (level 2).
- **Spacing:** A normal level of whitespace is applied within and between UI elements (level 2).
